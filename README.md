# iGaming Analytics Dashboard (Sportsbook & Casino Performance)
This project analyzes iGaming platform data across multiple countries to understand customer behavior, revenue performance and trends across Sportsbook and Casino segments over time.

## Tools Used
- Excel (Data Cleaning)
- Tableau (Dashboard)
- PowerPoint (Presentation)

## Key Insights
- Revenue distribution across countries and time periods
- Performance comparison between Sportsbook and Casino segments
- Customer behavior based on sign-ups and active users
- Deposit patterns and their impact on revenue
- Identification of high and low performing regions

## Files Included
- Raw and cleaned dataset (country + time-based data)
- Tableau dashboard
- Presentation slides
